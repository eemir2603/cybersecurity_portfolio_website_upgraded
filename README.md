# Junior Cybersecurity Engineer Portfolio

Welcome to my personal cybersecurity portfolio repository! This project is a modern, responsive, and dark-themed portfolio website built to showcase my skills, projects, and learning journey in the cybersecurity field.

## Overview

As an aspiring Security Engineer, I needed a centralized place to document my tools, methodologies, and homelab experiments. This website acts as my digital resume and a public log of my continuous learning in areas like network reconnaissance, automation, and penetration testing.

## Tech Stack

- **HTML5:** Semantic structure and accessibility.
- **CSS3:** Custom styling, Grid/Flexbox layouts, and modern UI/UX animations (no external frameworks used).

## Features

- **Dark "Cyber" Aesthetic:** Clean, terminal-inspired design without being overly cliché.
- **Project Showcase:** Detailed sections for tools built with Python and Bash, complete with terminal mockups and GitHub links.
- **Active Learning Logs:** A dedicated section to share current certification progress (e.g., eJPT) and tools I am currently mastering (e.g., Burp Suite).
- **Fully Responsive:** Optimized for both desktop and mobile viewing.

##  Installation & Usage

You don't need any complex build tools to run this site locally.
